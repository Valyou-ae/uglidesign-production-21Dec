=== VALIDATION LOGIC ===

Next button enabled when:
- environmentPrompt.length > 5 (more than 5 characters)

Next button disabled when:
- environmentPrompt is empty or <= 5 characters

=== STATE MANAGEMENT ===

State variable:
- environmentPrompt: string (initially empty "")

Template click:
- Sets environmentPrompt to template.prompt value

Textarea change:
- Updates environmentPrompt with textarea value

Selection logic:
- Template is visually selected if environmentPrompt === template.prompt
- If user modifies text, template deselects (no match)

=== KEYBOARD NAVIGATION ===

Enter key:
- If validation passes, proceed to next step (Colors)
- If in textarea, allow newline (don't proceed)

Escape key:
- Go back to previous step (Style)
