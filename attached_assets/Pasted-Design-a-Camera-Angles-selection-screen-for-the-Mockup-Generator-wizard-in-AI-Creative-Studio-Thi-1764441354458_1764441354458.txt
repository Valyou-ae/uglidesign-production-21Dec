=== VALIDATION LOGIC ===

Next button enabled when:
- selectedAngles.length > 0 (at least one angle selected)

Next button disabled when:
- selectedAngles is empty (length === 0)

=== STATE MANAGEMENT ===

State variables:
- selectedAngles: MockupAngle[] (array of angle IDs)
- Initially set to: ['front'] (front view pre-selected)

- selectedColors: ProductColor[] (may be empty since Colors step is now after Angles)
- selectedPlatform: string (used to conditionally show Size Chart)

Toggle behavior:
- If angle is selected, clicking removes it from array
- If angle is not selected, clicking adds it to array
- Multiple angles can be selected simultaneously

Select All:
- Sets selectedAngles to all available angle IDs
- If platform is Amazon: ['front', 'three-quarter', 'side', 'closeup', 'size-chart']
- If platform is NOT Amazon: ['front', 'three-quarter', 'side', 'closeup']

Clear All:
- Sets selectedAngles to empty array []

Standard Pack:
- Sets selectedAngles to ['front', 'three-quarter']

Full Collection:
- Same as Select All

isAllSelected:
- true if selectedAngles.length === allAvailableAngles.length

=== CONDITIONAL LOGIC ===

Size Chart visibility:
- Only show Size Chart angle card when selectedPlatform === 'amazon'
- Hide completely for all other platforms (etsy, shopify, general, etc.)

Available angles calculation:
- Filter MOCKUP_ANGLES to exclude 'size-chart' if platform !== 'amazon'
- This affects:
  - Grid display (4 or 5 cards)
  - Select All functionality
  - Full Collection button count
  - isAllSelected calculation

Mockup Counter logic (Updated for new step order):
- Since Colors step is now AFTER Angles:
  - Option A: Hide counter completely on Angles step (show only on Colors step)
  - Option B: Show only angles count: "{selectedAngles.length} Angles selected"
  - Option C: Keep counter but show 0 for colors until selected

=== KEYBOARD NAVIGATION ===

Enter key:
- If validation passes (at least 1 angle selected), proceed to next step (Scene)
- Blocked if in textarea/input (not applicable on this screen)

Escape key:
- Go back to previous step (Style)
